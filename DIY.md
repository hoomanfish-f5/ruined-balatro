# Build Your Own
here how to build your own ruined-balatro fork

download zip from repo or fork them

or using git 

once you download extract them

also extract "balatro_diy_build.zip"
# Build APK
to actually build own and test using Android Studio 

run Android Studio

go to File > New > Project from version control

Paste github repo then choose perfer to clone 

and select "Clone"

also need to extract another zip content to build

prompt will ask you to to trust project just simply press "Trust project"

this will initiate gradle sync 
#Sign APK
on menu toolbar click "Build"

go to "Build Bundle (s) / APK (s)"

then press "Build APK"

if was completed goto project folder and find apk file (depend where location you install)
# Done
you have complete sign apk to test if work 

use your phone or android emulator to install and run 

or you can edit file from en-us.lua (other language not supported)

Good luck build your own APK :)
